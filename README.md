notionsync
=======


notionsync is a CLI for https://www.notion.so/ja-jp

## Synopsis

```console
# set $TODO environment variable before using

% notionsync pull
...

% notionsync push 
```

## Description

Notion client to edit markdowns locally. It download markdowns with frontmatter.

## Installation

### Homebrew

```console
% brew install 
```

### go get

```console
% go get 
```
